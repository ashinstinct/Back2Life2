Prepare Your Reference Photo

A clear photo is the foundation this whole generation is built on

1

Choose a clear, well-lit photo of yourself (or whoever's appearing in the clip). Front-facing works best — avoid sunglasses or heavy shadows.

2

Turn it into a multi-angle reference sheet using GPT Image 2 — four angles plus a face close-up on a plain background, exactly like the one below. This is the same method used to build the reference sheet for this generation.

3

Open Dreamina AI and start a new Seedance 2 Mini image-to-video generation.

4

Upload your reference sheet as the reference / character image.

3

Paste The Prompt & Generate

A recreation prompt written from this exact generated clip

1

Copy the prompt below exactly as written and paste it into the Seedance 2 Mini prompt field.

2

Swap in your own details anywhere you want a different look — clothing, setting, colors — the structure is what matters most.

3

Tap Generate and wait for the render.

📋 Recreation Prompt — Written From This Generation
Create a futuristic black and neon colored special edition BACK2LIFE x Nike collab shoe from multiple angles like a product spec sheet, using the colours from the back2life.ai logo attached. The back of the shoe should have text:

'BACK2LIFE 
x 
Nike'

---

Advert prompt (used with the finished spec sheet as reference):

Create a cinematic advert for these new Nike Air x Back2Life.ai collaboration shoes.
🎬

This is a two-stage workflow: run the spec-sheet prompt first with your logo attached as a reference image, then feed the finished spec sheet into Seedance 2 Mini with the short advert prompt. The advert prompt is deliberately short — the spec sheet is doing most of the work as the visual reference, so Seedance mainly needs to know it's building a commercial, not a full scene description.

4

Download & Share

Once generation completes, preview the clip. If it isn't quite right, regenerate — results vary between runs.

2

Tap Download to save the MP4 — ready for TikTok, Instagram Reels, and YouTube Shorts.
